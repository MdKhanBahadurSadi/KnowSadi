GhorBondhu – Local Service Booking App

"GhorBondhu" is a user-friendly mobile application created to connect people within a community by making local service booking easier, faster, and more organized.

Flutter / Dart Firebase
  • Secure Login System: Email/Password authentication with **Email Verification** for a trusted user base.
  • Dynamic Newsfeed: Real-time feed to browse services, including title, description, price, and contact info.
  • Service Posting Form: Allows any user to easily post a service they are offering.
  • Order Now Feature: Instant request mechanism for a smooth and interactive booking process.

✅ Status: MVP completed, includes splash screen, auth system, and live data loading.

GhorBondhu Website – Digital Service Platform

The web version of the GhorBondhu app, a responsive platform to help people find or offer everyday services within their community.

HTML CSS PHP
  • Email Login with Verification: Secure sign-up and login process.
  • Live Newsfeed of Services: Homepage displays real-time service posts.
  • Responsive Design: Works smoothly on desktop and mobile devices.
